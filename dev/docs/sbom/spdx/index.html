    SPDX

    Trivy generates reports in the SPDX format.

    You can use the regular subcommands (like image, fs and rootfs) and specify spdx with the --format option.

    $ trivy image --format spdx --output result.spdx alpine:3.15
    Result 
    $ cat result.spdx
